# Measurement

Use this when the client asks how to evaluate GEO site architecture work after implementation.

## Primary Measurement

- Search Console generative AI performance reports, if available for the property.
- Search Console page impressions, queries, countries, devices, and date trends.
- Pages appearing in AI features, where available.
- Crawl/indexing status for optimized pages.

## Secondary Measurement

- Manual AI answer checks for priority buyer questions.
- Brand inclusion in comparison, recommendation, trust, pricing, and setup answers.
- Citation/source changes in Dageno or other monitoring platforms.
- Referral traffic from AI/search surfaces where visible.
- Conversion path changes on optimized pages.

## Reporting Rules

- Do not promise guaranteed AI rankings, citations, or inclusion.
- Separate monitoring-sample data from total-market claims.
- Keep prompt/model/region/date windows consistent when comparing periods.
- For regulated industries, measure trust/risk answer quality in addition to visibility.
